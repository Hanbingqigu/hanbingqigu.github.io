Description

给定 nnn 个函数形如 fi(x)=ai∣x∣+bix+cif_i(x)=a_i|x|+b_ix+c_ifi​(x)=ai​∣x∣+bi​x+ci​ 和 x0x_0x0​。求 nnn 阶排列 ppp，最大化 F(p)=fp1(fp2…fpn(x0))F(p)=f_{p_1}(f_{p_2}\dots f_{p_n}(x_0))F(p)=fp1​​(fp2​​…fpn​​(x0​)) 。

n,∣ai∣,∣bi∣,∣ci∣≤15n,|a_i|,|b_i|,|c_i| \le 15n,∣ai​∣,∣bi​∣,∣ci​∣≤15。

Solution

已经轻松最优解。

我们充分发扬人类智慧，参考经典模拟退火解决 TSP (最短哈密顿回路问题) ，设当前的排列为 ppp，每次向邻域尝试"走一步"。根据 TSP 的经验，定义邻域结构如下：

  • 随机 i,ji, ji,j，交换 pi,pjp_i,p_jpi​,pj​。
  • 随机 l,rl,rl,r，翻转 pl,pl+1,…,prp_l,p_{l+1},\dots,p_rpl​,pl+1​,…,pr​。
  • 随机 i,ji,ji,j，将 pip_ipi​ 插入至位置 jjj。

模拟退火时，每次尝试走一步至 p′p'p′，令 ΔE=F(p′)−F(p)\Delta E= F(p')-F(p)ΔE=F(p′)−F(p)，当前温度为 TTT，则接受新解的概率为：

P={1if   ΔE>0eΔETotherwise.P = \begin{cases} 1 &\text{if } \; \Delta E > 0 \\ e^{\frac {\Delta E} T} &\text{otherwise.} \end{cases} P={1eTΔE​​if ΔE>0otherwise.​

注意 ΔE\Delta EΔE 的符号，eee 的指数上一定是个负数，不然就会跟我一样永远接受最劣解导致赛时过不去以为假了。

由于 nnn 非常小，温度和降温系数都可以极限一点，我这里取初始温度 T0=109T_0=10^9T0​=109，终止温度 Tk=10−5T_k = 10^{-5}Tk​=10−5，降温系数 d=0.995d = 0.995d=0.995。为了得到更优质的解在每次退火完随机取邻域内 100010001000 个点取最大值，最后再卡一下时多跑几次就能直接过了。（事实上不需要这么极限的参数，只是由于我赛时写假了以为过不了，参数调小一点随便最优解）。
